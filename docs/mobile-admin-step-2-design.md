# Мобильная админ-панель — Шаг 2. Проектирование навигации и карточных шаблонов

## 1. Навигация
- **AppBar (верхняя шапка)**
  - Кнопка меню (гамбургер) слева, логотип по центру на Dashboard.
  - Глобальный поиск (лупа) → открывает полноэкранный поиск по пользователям/школам/счетам.
  - Иконка уведомлений с бейджем, аватар администратора с выпадающим меню.
  - На разделах: заголовок + кнопка «создать» (или FAB), кнопка фильтров, контекстные действия.
- **Hamburger Drawer**
  - Разделы: Dashboard, Пользователи, Школы, Мастер-классы, Счета, Заявки, Чат, Настройки.
  - Бейджи: новые заявки, просроченные счета, непрочитанные сообщения.
  - Быстрые действия (иконки справа): «+ Пользователь», «+ Школа», «+ МК».
- **Bottom actions**
  - FAB для главного действия раздела (создать пользователя/школу/счет).
  - В режиме массового выбора появляется нижняя панель с действиями: «Назначить», «Удалить», «Экспорт», «Изменить статус».
- **Уведомления**
  - Отдельный экран (Drawer) с лентой событий: фильтр по типу (оплаты, заявки, чат).
  - Свайп по карточке уведомления → «прочитано», «выполнено».

## 2. Карточные представления списков
- **Пользователи**
  - Заголовок: ФИО + бейдж роли.
  - Подпись: школа/класс, контакт.
  - Контент: счетчик детей (для родителя), отметка наличия задолженности, дата последней активности.
  - CTA: кнопка «Детали» + контекстное меню (редактировать, удалить, reset access).
- **Школы**
  - Заголовок: название + город.
  - Подпись: контактное лицо, телефон.
  - Контент: список классов/групп, количество мастер-классов, активных заявок.
  - CTA: «Редактировать», меню (добавить МК, удалить, отправить письмо).
- **Мастер-классы (листинг)**
  - Заголовок: название + дата/время.
  - Подпись: школа, аудитория, статус.
  - Контент: количество детей, оплачено/ожидает, назначенный исполнитель.
  - CTA: «Открыть», меню (редактировать, дублировать, отменить, отправить отчёт).
- **Мастер-класс (деталь)**
  - Хедер: статус, кнопки «Отметить оплату», «Прислать ссылку», «Назначить исполнителя».
  - Внутренние секции: участники (карточки), счета (карточки), заметки, материалы.
  - На мобильном: табы превращаются в верхний scroll или segmented control.
- **Счета**
  - Заголовок: номер счета + сумма.
  - Подпись: родитель, мастер-класс.
  - Контент: статус (бейдж), дата оплаты/создания, тип оплаты, список детей.
  - CTA: «Просмотреть», меню (отправить повторно, отменить, отметить наличными).
- **Заявки**
  - Заголовок: школа + класс/группа.
  - Подпись: контакт, статус обработки.
  - Контент: дата проведения, пожелания, ответственный менеджер.
  - CTA: «Открыть», меню (назначить, завершить, перенести).
- **Чат**
  - Лист диалогов: ФИО, роль, последнее сообщение, бейдж непрочитанных.
  - Сообщения: bubble view, swipe для действий, нижняя панель ввода с кнопкой вложений.

## 3. Фильтры и массовые действия
- **Быстрые фильтры в AppBar**: горизонтальный список pill-buttons (статус, роль, дата).
- **Расширенные фильтры**: bottom sheet / Drawer с чекбоксами, радиокнопками, date-range picker.
- **Массовый выбор**: long-press → чекбоксы на карточках, появляется нижняя панель действий, FAB превращается в «Отмена».

## 4. Взаимодействие
- **Свайпы**: влево/вправо — быстрые действия (архив, позвонить, отметить оплачено).
- **Аккордеоны**: раскрывать дополнительные данные (примечания, история оплат).
- **Bottom sheets**: краткие детали/редактирование без перехода на отдельный экран.
